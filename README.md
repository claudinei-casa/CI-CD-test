# Projeto Hello World CI/CD


Este é um projeto simples em Python para demonstrar um fluxo de Git e CI/CD.

## Como rodar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o script:
   ```bash
   python hello.py
   ```

3. Execute os testes:
   ```bash
   pytest
   ```
